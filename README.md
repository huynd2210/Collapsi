Collapsi
========

AO* solver for Collapsi with SQLite caching and a simple Flask web UI.

Quickstart
----------

1) Python 3.10+
2) Install deps:
```
pip install -r requirements.txt
```
3) CLI solver (caches to SQLite):
```
python game.py --size 4 --db collapsi.db --seed 1
```
4) Web UI:
```
python app.py
# open http://127.0.0.1:5000
```
